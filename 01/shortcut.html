[Top 15 IntelliJ IDEA shortcuts]


[Top 15 IntelliJ IDEA shortcuts] 를 바탕으로 유용한 IntelliJ IDEA 단축키 15개를 정리해 보았습니다.

  1. ⌥ + enter : Suggestions for error correction



    2. - 코드에서 오류가 발견된 경우 해당 오류 위에 커서를 놓고 ⌥ + enter를 누르면 문제에 대한 수정 제안 목록이 표시됩니다.

  2. F2 : Navigate errors




    3. - 오류 메시지에서 F2를 누를 경우 해당 오류로 바로 이동할 수 있습니다.

  3. ⌘ + 1 : Open project window



    4. - 평소에 전체 화면으로 코드 에디터를 사용한다면 ⌘ + 1을 눌러 프로젝트 창으로 이동할 수 있습니다.

  4. Esc : Focus back to editor



    5. - Esc를 누르면 에디터로 돌아와 다시 코드에 집중할 수 있습니다.

  5. ⌘ + E : View recent files



    6. - 이 단축키를 누르면 최근 파일 대화상자가 열려 방향키로 파일을 탐색할 수 있습니다.

  6. ⌘ + B : See declaration of symbol



    7. - 필드에서 단축키를 누르면 커서가 필드 선언으로 이동합니다.

  7. ⌥ + F7 : See usage



    8. - 단축키를 누르면 필드 선언 또는 인터페이스를 구현한 클래스를 비롯하여 해당 인터페이스가 사용된 모든 위치가 검색창에 표현됩니다.

  8. ⌃ + ⌃ : Run anything



    9. - 최근 실행한 실행 구성 목록이 표시됩니다.

  9. ⌥ + ↑, ⌥ + ↓ : Expand / Close selection



    10. - 커서 주변의 코드 섹션을 확장하거나 축소할 수 있습니다.

  10. ⌘ + /, ⌘ + ⌥ + / : Comment / Comment block



    11. - 단축키를 누르면 코드가 주석으로 처리됩니다.

  11. ⌘ + ⇧ + enter : Auto complete



    12. - 세이콜론을 추가하고 괄호를 닫는 등 자동으로 구문을 완성해줍니다.

  12. ⌘ + ⌥ + L : Reformat file



    13. - 현재 파일의 서식을 간편하게 프로젝트 표준에 맞춥니다.

  13. ⌃ + T : Refactoring



    14. - 적용 가능한 리팩토링 옵션이 표시됩니다.

  14. ⌘ + ⇧ + A : Find action



    15. - IntelliJ IDEA의 모든 액션을 겁색할 수 있습니다.

  15. ⇧ + ⇧ : Search



    16. - 무엇이든 검색이 가능한 검색 창이 표시됩니다.
