[Intellij IDEA Ultimate Edition 설치하기 (maxOS Big Sur)]


기존에 JAVA와 Spring Framework를 공부하며 전공 시간에 사용하던 Eclipse를 벗어나 IntelliJ IDEA를 설치하게 되었다.
설치하며 학생인증을 받아 무료로 Ultimate 에디션을 사용하는 방법과 패키지 관리 매니저를 사용하여 쉽고 간편하게 설치하는 방법을 정리해보았다.

유료와 무료 에디션에는 위 사진과 같은 기능 차이들이 존재한다.
언뜻 보기에도 많은 차이가 존재하는데, 'Intellij를 사용한다!' 라고 말하기 위해선 Ultimate Edition을 사용해야 한다고 한다.

하지만 학생에게는 너무 비싼 가격....

그래서 학생들에게는 학생인증을 통하여 무료로! 이러한 기능들을 제공한다.

Intellij Ultimate Edition 학생인증 받기

- 먼저 아래 링크로 접속한다
학생인증 링크
- 스크롤을 내리다 보면 아래와 같은 화면을 볼 수 있는데, apply now를 클릭한다.

- 클릭하면 아래와 같은 입력창이 나오는데, Github Student Development Pack이 이미 있다면 바로 인증이 끝난다.
- Github Student Development Pack은 등록해두면 많은 혜택을 볼 수 있으니 미리미리 등록해 두는것을 추천한다.

- 위 양식을 다 채우고, 이메일 인증이 끝나면 등록이 완료된다.


Intellij Ultimate Edition 무료로 설치하기

- 내장 Terminal이나 iTrem2를 열고 다음의 한 문장만 입력해주면 설치가 시작된다.
brew install --cask intellij-idea
- 다음과 같이 설치가 진행되는 것을 알 수 있다.

- 설치가 완료되면 다음과 같은 안내문이 출력된다.
- 런치패드에 다음과 같은 아이콘이 생긴것을 확인할 수 있다.

- 실행시킨 후 임의의 프로젝트를 생성해보자
- Tools 탭의 Create Command-line Launcher를 클릭한다.

- OK를 눌러 다음으로 넘어간다.

- 터미널이나 iTerm2를 다시 켜고, 원하는 경로에서 idea .을 입력하면 해당 폴더에서 IntelliJ가 실행된다
